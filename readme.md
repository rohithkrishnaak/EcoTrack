# 🌿 EcoTrack

**A solution for Hack2skill PromptWars Challenge 3**

## 📖 Problem Statement
*Design a solution that helps individuals understand, track, and reduce their carbon footprint through simple actions and personalized insights.*

## 🚀 About The Project
EcoTrack is a lightweight, intuitive web application designed to help users visualize the environmental impact of their daily routines. By simply inputting their daily transportation miles, heavy electricity usage, and dietary habits, the app calculates an estimated CO₂ emission score.

More importantly, it dynamically identifies the user's heaviest emission category and generates **personalized, actionable insights** to help them make greener choices the following day.

## ✨ Features
* **Simple Tracking:** Clean, friction-free input form for daily habits.
* **Instant Calculation:** Real-time conversion of daily activities into kg of CO₂.
* **Personalized Insights:** An algorithm that analyzes the data array and provides specific tips based on the user's highest emission source.
* **Fully Responsive:** Accessible and visually appealing on both desktop and mobile screens.

## 🛠️ Technology Stack
* **HTML5:** Structure and semantics.
* **CSS3:** Styling, custom properties (variables), and responsive layout.
* **Vanilla JavaScript:** Logic, calculation algorithms, and DOM manipulation. 

## ⚙️ How to Run Locally
Because this project uses raw HTML/CSS/JS with no backend dependencies, running it is incredibly simple:
1. Clone this repository to your local machine.
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Start tracking your footprint!
